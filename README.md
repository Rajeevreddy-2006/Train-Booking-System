A console-based Train Booking System built using C++ and Object-Oriented Programming principles.
The system supports ticket booking, cancellation, authentication, fare calculation, and
persistent storage using files.

Features:
- User and Admin authentication
- Train routes with multiple stations
- Date-wise seat allocation
- Waiting list handling
- Rule-based fare calculation (base fare, train charge, GST, discount)
- Ticket booking and cancellation
- File-based data persistence

Tech Stack:
- C++
- Object-Oriented Programming
- STL
- File Handling
- Mutex for safe booking

How to Run:
1. Compile: g++ main.cpp -o train
2. Run: ./train
